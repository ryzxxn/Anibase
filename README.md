## Getting Started

### Prerequisites

- MyAnimeList account: [Create an account](https://myanimelist.net/register.php) if you don't have one.
- MyAnimeList API Key: [Generate an API Key](https://myanimelist.net/apiconfig) from your MyAnimeList account.
